Power up
Power source is provided by MSP430FR6989.
CC2640R2F can be directly power up by MSP430FR6989's 3V pin.
MSP430FR5994 need to power up by power cable first,
then connect to MSP430FR6989's 3V pin, 
then disconnect power cable.

Server: Raspberry Pi
BLE Library: gattlib
Link: https://github.com/labapart/gattlib
